Colección personal de símbolos de uso frecuente —flechas, letras griegas, operadores matemáticos, teoría de conjuntos, comillas, signos de puntuación, etc.— con su código HTML, pensada para tenerlos siempre a mano.

## Ver online

La página está pensada para visualizarse mediante **GitHub Pages**:

**https://vginer.github.io/misc-symbols/**

## Uso

- Los símbolos están **agrupados por categoría**.
- **Clic** sobre un símbolo → copia el **símbolo**.
- **`Mayús` + clic** → copia su **código HTML** (por ejemplo, `&rarr;`).
- Diseño **responsive**: se adapta a móvil y escritorio.

## Contenido del repositorio

- **`docs/index.html`** — el catálogo; página HTML autónoma que se renderiza en GitHub Pages.

## Uso local

Descarga `index.html` del directorio `docs` y ábrelo en el navegador. *(El copiado al portapapeles requiere abrir la página vía `https` —por ejemplo, en GitHub Pages— o como archivo local en el navegador.)*
